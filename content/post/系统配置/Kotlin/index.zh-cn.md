# Kotlin

> 下载[链接](https://github.com/JetBrains/kotlin/releases)

环境变量在path上添加如下内容

```
D:\env\kotlinc\bin
```

将`D:\env\kotlinc`替换成自己的安装路径

查看信息

```sh
kotlinc -version
```
